Getting Started with Annotorious

Annotorious is a JavaScript library for image annotation. Add drawing, commenting and labeling functionality to images on your website with just a few lines of code. Use the JavaScript API to integrate Annotorious with your own applications, or customize and extend it with your own styles and add-on components.

Click the annotation below to edit. Click and drag the mouse to create a new annotation.
