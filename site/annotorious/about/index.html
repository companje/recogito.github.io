About Annotorious

Annotorious is a modernized reboot of my outdated original Annotorious, which I’m no longer supporting.

License

Annotrious is licensed under the terms of the BSD 3-Clause license. That basically means: feel free to use this code in whatever way you wish, for non-commerical and commercial projects. But keep the attribution/license file included. And bear in mind: this code comes as is. You get it for free, if breaks something, please report an issue, but don’t complain to us ;-)

Need Help?

Get in touch via our Annotorious Gitter chat. If you encounter bugs, or have feature requests, please report them via our GitHub issue trackers:

  • annotorious/issues for the Annotorious standard version
  • annotorious-openseadragon/issues for the OpenSeadragon plugin

Source Code

All source code is available on our recogito GitHub organization, hosted in the following repositories:

  • annotatorious for the standard version
  • annotatorious-openseadragon for the OpenSeadragon plugin
  • recogito-client-core for shared base components
